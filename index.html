About The Project
  • This is a demo application of the W3 organisation for speech API with the hope to find useful application for the people with disabilities.

  • First section consists of speech to text application which will help people having difficulty using their hands in typing on the keyboard.

  • The second section consists of text-to speech synthesis application which will be helpful for people with speech disorders and people with visual impairment.
